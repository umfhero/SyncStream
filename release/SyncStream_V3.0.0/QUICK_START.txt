## ğŸ¯ How to Use

### System Tray Features (NEW!)

**Minimize to Tray:**
- Click the **X** button to close
- App minimizes to system tray (doesn't quit)

**Open from Tray:**
- **Left-click** icon â†’ Opens in **Compact Mode** (small window)
- **Right-click** icon â†’ Menu with options:
  - Open in Compact Mode
  - Show in Normal Mode
  - Hide
  - Exit (fully quit)

### Compact Mode vs Normal Mode

**Compact Mode** (380x295 window):
- Small, streamlined interface
- Perfect for quick file sharing
- Essential controls only
- Quick drag-and-drop

**Normal Mode** (full window):
- All features available
- Statistics, settings, profile manager
- Gallery view with search and filters
- Full file history

**Toggle:** Click the arrow button in top bar to switch modes

### Windows Startup (NEW!)

**Enable Auto-Start:**
1. Go to **Settings**
2. Find "Startup:" section (top left)
3. Check "Run SyncStream when Windows starts"
4. App will start automatically with Windows (minimized to tray)

**Disable:**
- Uncheck the same box in Settings

## ğŸ’¡ File Sharing Basics

### Sending Files

**Method 1: Drag & Drop**
1. Connect to a peer
2. Drag files from Windows Explorer
3. Drop anywhere on the window
4. Files send automatically

**Method 2: Browse**
1. Click **Browse Files** button
2. Select files
3. Files appear in gallery

**Method 3: From Gallery**
1. Click **Gallery** button
2. Click any shared file
3. Select "Send to Peer"

### Receiving Files

Files are automatically saved to:
```
%APPDATA%\SyncStream\Downloads\
```

## ğŸ› Troubleshooting

### System Tray Icon Not Showing
1. Restart SyncStream
2. Check if `pystray` is installed: `pip list | findstr pystray`
3. Look in system tray overflow (hidden icons)

### Can't Connect to Peer
1. Check Tailscale is running: `tailscale status`
2. Verify peer's IP address
3. Ensure both devices on same Tailscale network

### Files Not Sending
1. Check connection status (LED should be green)
2. Try reconnecting
3. Check file permissions

### Startup Not Working
1. Check Windows Task Manager â†’ Startup tab
2. Verify "SyncStream" is listed
3. May require restart to take effect
